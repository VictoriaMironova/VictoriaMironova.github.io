Objects

Об’єкти

In JavaScript, almost "everything" is an object. В JavaScript, майже "все" є об'єктом 

    Booleans can be objects (or primitive data treated as objects)
    Numbers can be objects (or primitive data treated as objects)
    Strings can be objects (or primitive data treated as objects)
    Dates are always objects
    Maths are always objects
    Regular expressions are always objects
    Arrays are always objects
    Functions are always objects
    Objects are objects
    In JavaScript, all values, except primitive values, are objects.

    Primitive values are: strings ("John Doe"), numbers (3.14), true, false, null, and undefined.

Objects are Variables Containing Variables Об'єкти це змінні, що містять змінні

JavaScript variables can contain single values Змінні JavaScript можуть містити поодинокі значення 


                var person = "John Doe";

Objects are variables too. But objects can contain many values.

The values are written as name : value pairs (name and value separated by a colon). Об'єкти є змінними теж. Але об'єкти можуть містити безліч значень. Значення записуються у вигляді: ім'я: значення (ім'я і значення через двокрапку). 


                var person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};

The named values, in JavaScript objects, are called properties. Названі значення, в об'єктах JavaScript, називаються властивостями.


                        var person = {
            firstName:"John",
            lastName:"Doe",
            age:50,
            eyeColor:"blue"
                        };
Property Value
firstName John

eyeColor blue

Accessing JavaScript Properties Доступ до властивостей JavaScript

The syntax for accessing the property of an object is: 
objectName.property          // person.age
or 
objectName["property"]       // person["age"]

Example

concole.log(person.firstname + " is " + person.age + " years old.");
person["firstname"] + " is " + person["age"] + " years old.";

JavaScript for...in Loop

The JavaScript for...in statement loops through the properties of an object. JavaScript вираз for...in в циклі перебирає всі властивості об'єкта 
for (variable in object) {
    code to be executed
}

Example

var txt = "";
var person = {fname:"John", lname:"Doe", age:25};
var x;
for (x in person) {
    txt += person[x] + " ";
}
document.getElementById("demo").innerHTML = txt;

Deleting Properties Видалення властивостей

The delete keyword deletes a property from an object: Ключове слово delete видаляє властивість 
var person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};
delete person.age;   // or delete person["age"];
The delete keyword deletes both the value of the property and the property itself. Ключове слово delete видаляє значення властивості та саму властивість

Creating a JavaScript Object Створення JavaScript об’єкту

1. Using an Object Literal Використовуючи літерали об’єкту

This is the easiest way to create a JavaScript Object.
Using an object literal, you both define and create an object in one statement. 

                    var person = {
    firstName:"John",
    lastName:"Doe",
    age:50,
    eyeColor:"blue"
};

2. Using the JavaScript Keyword new Використовуючи ключове слово new

The following example also creates a new JavaScript object with four properties: 

                    var person = new Object();
person.firstName = "John";
person.lastName = "Doe";
person.age = 50;
person.eyeColor = "blue";

Constructors

Конструктори

3. Using an Object Constructor Використовуючи конструктор об’єктів

The examples above are limited in many situations. They only create a single object. Sometimes we like to have an "object type" that can be used to create many objects of one type.
The standard way to create an "object type" is to use an object constructor function: 

                function Person(first, last, age, eye) {
                this.firstName = first;
                this.lastName = last;
                this.age = age;
                this.eyeColor = eye;
                }
var myFather = new Person("John", "Doe", 50, "blue");
var myMother = new Person("Sally", "Rally", 48, "green");

The this Keyword Ключове слово this

In JavaScript, the thing called this, is the object that "owns" the JavaScript code. В JavaScript, в те, що називається this, є об'єктом, який "володіє" кодом JavaScript.

The value of this, when used in a function, is the object that "owns" the function. Значення this, при використанні в функції, є об'єктом, який "володіє" функцією.

The value of this, when used in an object, is the object itself. Значенням this, при використанні в якості об'єкта, є сам об'єкт.

Adding New Methods in Constructors Додавання нових методів в конструкторах


function Person(firstName, lastName, age, eyeColor) {
    this.firstName = firstName;
    this.lastName = lastName;
    this.age = age;
    this.eyeColor = eyeColor;
    this.changeName = function (name) {
        this.lastName = name;
    };
}
var myMother = new Person("Sally", "Rally", 48, "green");
myMother.changeName("Doe");

JavaScript Prototypes

Every JavaScript object has a prototype. The prototype is also an object. Кожний об’єкт JavaScript має прототип. Прототип є також об’єктом All JavaScript objects inherit their properties and methods from their prototype. Всі об'єкти JavaScript успадковують властивості і методи від їх прототипу. Objects created using an object literal, or with new Object(), inherit from a prototype called Object.prototype. Об’єкти створені задопомогою літералу чи конструкції new Object(), успадковують від прототипу Object.prototype Objects created with new Date() inherit the Date.prototype. Об’єкти створені з new Date() успадковують Date.prototype All JavaScript objects (Date, Array, RegExp, Function, ....) inherit from the Object.prototype. Всі JavaScript об’єкти успадковують від Object.prototype
